# LevelZ Refabricated

**Unofficial port of [LevelZ](https://modrinth.com/mod/levelz) by [Globox_Z](https://modrinth.com/user/Globox1997) — ported to Minecraft 1.21.11 with Mojang Mappings (Fabric).**

All credit for the original mod goes to **Globox_Z**.


![1](https://cdn.modrinth.com/data/cached_images/84748dc0bbf7c7d2f50bec09897b5397b08f97a0.jpeg)

## What's changed in this port
- Ported to Minecraft 1.21.11
- Migrated from Yarn to Mojang Mappings
- Adapted to work correctly with Minecraft 1.21.11 changes

## Requirements
- [Fabric API](https://modrinth.com/mod/fabric-api)
- [Cloth Config API](https://modrinth.com/mod/cloth-config)
- [LibZ Refabricated](https://modrinth.com/mod/libz)
